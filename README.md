# Client and Transaction Management System
*Course Project for "Programming Advices" Online Course, 2025*

## Description
This is a C++ console application designed to manage clients, transactions, and basic user accounts. 
Data is stored in `.txt` files with simple encryption. 
The system features a menu-driven interface and implements **user permissions using binary numbers** to control access to certain functionalities.

## Features
- Add, update, delete, and search clients
- Handle financial transactions
- User accounts with permissions using binary numbers
- Menu-driven interface for easy navigation
- Basic encryption for sensitive data
